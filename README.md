# S.O.S. Counseling - Specialized Outpatient Services

A modern, responsive website for S.O.S. Counseling in Grand Island, Nebraska.

## www.specializedoutpatientservices.com

### About

S.O.S. Counseling (Specialized Outpatient Services) provides compassionate mental health counseling for at-risk youth, families, and adults. Founded in 2014, they specialize in ADHD, anxiety, depression, PTSD, and family therapy.

### Features

- Responsive design (mobile, tablet, desktop)
- Modern CSS animations and transitions
- SEO optimized with JSON-LD structured data
- Contact form
- Team profiles with photos
- Services showcase
- Testimonial slider

### Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Open Sans)
