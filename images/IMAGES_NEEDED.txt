IMAGES NEEDED FOR COMPLETE SEO SETUP
=====================================

Please create/add the following images:

1. favicon.ico (16x16, 32x32 multi-size ICO file)
   - Place in root directory: /favicon.ico

2. favicon-16x16.png (16x16 PNG)
   - Place in: /images/favicon-16x16.png

3. favicon-32x32.png (32x32 PNG)
   - Place in: /images/favicon-32x32.png

4. apple-touch-icon.png (180x180 PNG)
   - Place in: /images/apple-touch-icon.png

5. icon-192.png (192x192 PNG for PWA)
   - Place in: /images/icon-192.png

6. icon-512.png (512x512 PNG for PWA)
   - Place in: /images/icon-512.png

7. og-image.jpg (1200x630 recommended for social sharing)
   - Place in: /images/og-image.jpg
   - This image appears when the site is shared on Facebook, Twitter, LinkedIn, etc.
   - Should include the S.O.S. logo, business name, and tagline

8. logo.png (transparent PNG logo)
   - Place in: /images/logo.png


RECOMMENDED DESIGN:
- Use maroon (#8B1A1A) as the primary color
- Include the S.O.S. text/branding
- Keep it simple and recognizable at small sizes

You can generate these using tools like:
- Canva (canva.com)
- Figma (figma.com)
- RealFaviconGenerator (realfavicongenerator.net)
